# Windows

#### 1.[下载Windows客户端](/README?id=客户端下载地址)安装完成后，运行 V2rayn.exe，点击 「订阅」-「订阅设置」-「添加...」； ####

![win](/_assets/images/winodws/win1.png)

#### 2.在网站用户中心里，复制节点订阅链接；粘贴节点订阅链接至「URL」中，点击「确认」，然后关闭返回到主界面； ####

![win](/_assets/images/winodws/win2.png)

#### 3.点击「订阅」-「更新订阅」； ####

![win](/_assets/images/winodws/win3.png)

#### 4.选择一个延迟较低的节点； ####

![win](/_assets/images/winodws/win4.png)

#### 5.「HTTP代理」一般选择「PAC模式」，适合日常上网，如不能使用，则切换「全局」即可； ####

![win](/_assets/images/winodws/win5.png)

#### 最后打开浏览器Google即可科学上网。 如果速度较慢，可以却换节点，到最满意 ####
